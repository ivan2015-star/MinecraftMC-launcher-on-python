# MinecraftMC-launcher-on-python
A Minecraft launcher I made myself using Pyinstaller.

FAQ:

How to: How to specify a custom installation path?

1. Click Settings
2. Specify the Minecraft directory
3. Go back to Play
4. Specify installation path ((There is an install to: there you specify the location where the version is installed))
5. Once the version is installed, select it and press play.
